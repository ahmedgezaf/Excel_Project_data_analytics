# 📊 Data Jobs Market Intelligence Dashboard

> **Excel Power Pivot Project** · Built as part of [Luke Barousse](https://www.youtube.com/@LukeBarousse)'s *Excel for Data Analytics* course

---

## 🚀 What I Built & Why

I built an **interactive market intelligence dashboard** in Excel that answers a question every early-career data professional asks:

> *"Which role should I target, and which skills should I learn to maximise my salary?"*

Using **Power Pivot** and **Pivot Tables**, I pulled together a dataset of **167,000+ real job postings** and turned raw pivot output into a clean, decision-ready dashboard — the kind you'd actually show to a recruiter or a hiring manager.

## 💡 Key Findings

- **SQL & Python** appear in **57%** of all data job postings — they're non-negotiable
- **Senior Data Scientist** tops salary charts at **$155,000** median
- **Spark** commands the highest median skill-salary at **$140,000**
- **US salaries** can be **~20–48% higher** depending on the role (biggest gap: ML Engineer)
- **Data Engineers** require the most skills per posting (~8.1 avg) — the most demanding generalists
- More skills ≠ higher pay: Cloud Engineer requires ~4.8 skills but earns only $90K
